# Okechuwku Emeka Joel — Academic Portfolio & Student Management Platform

A responsive personal academic portfolio website designed to present a university student's profile, skills, academic achievements, projects, and student-life planning tools in one polished experience.

## What’s inside

- **Home** — academic portfolio landing page with key focus areas
- **About** — academic background, goals, skills, and student interests
- **Projects** — showcase of the portfolio site and student tools
- **Planner** — interactive academic task manager with localStorage persistence
- **Contact** — collaboration and networking form with validation

## Tech stack

- HTML5
- CSS3
- JavaScript
- Browser localStorage

## Local preview

Open `index.html` in a browser, or use a local static server if preferred.

## GitHub Pages deployment

1. Push this folder to a GitHub repository.
2. In the repository settings, open **Pages**.
3. Choose the branch containing the site files and set the root folder.
4. Save, then use the generated Pages URL as your hosted link.

## Notes

- The planner stores tasks in the browser only.
- Contact form validation runs entirely on the client side.
- The site is designed to be easy to expand with new academic sections and tools.
